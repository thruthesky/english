#01

화상카메라, 헤드셋 준비

  • 추천 카메라

    화상카메라 설치

    화상카메라는 USB단자에 꼽기만 하면 드라이버가 자동 설치되는
    제품을 권해드립니다.
    화상카메라에 마이크가 내장되어 있는 경우 헤드셋의 마이크와 충돌을 일으켜
    에코현상이 발생할 수 있으니 수업시에서는 꼭 내장마이크를 꺼주시기 바랍니다.

  • 추천 헤드셋

    헤드셋 설치

    헤드셋도 USB단자에 꼽기만 하면 드라이버가 자동 설치되는 제품을 권해드립니다.
    저가품이어도 상관없으나 꼭 마이크가 달린 헤드셋을 구매해 주시기 바랍니다.
    만약 분홍색과 연두색이 잭이 달린 헤드셋을 구매하셨다면
    컴퓨터 뒷면이나 앞면에 위치한 잭에 꼽으시면 됩니다.

STEP
#02

화상강의실 설치 방법

상단의 화상프로그램설치 버튼을 클릭한 후 팝업창의 지시에 따라 프로그램을 설치하시면 됩니다.

크롬(Chrome)브라우저에서는 화상솔루션이 설치되지 않습니다. Window10 기본 브라우저인 Edge에서는 Internet Explorer 11을 실행한 후 설치해 주시기 바랍니다.
Internet Explorer 11 실행방법 : 윈도우 시작버튼 > 보조프로그램 > Internet Explorer 11 실행 > 화상솔루션 설치

  • 01. 화상솔루션설치 클릭

    홈페이지 우측 상단에 위치한 [화상강의실 설치]버튼를 클릭합니다.

  • 02.ActiveX 설치

    사이트 하단의 추가기능 설치를 클릭, 프로그램을 설치합니다.

  • 03.헤드셋 연결확인

    오디오/비디오 설정마법사를 진행합니다. 처음 헤드셋 연결확인창이 나오면
    상단의 [스피커시험]을 클릭하여 소리를 확인하고, 들리지 않으면
    [다른 스피커 장치]를 선택합니다.

  • 04.마이크동작 확인

    마이크장치를 선택한 후 상단의 [마이크시험]을 클릭하여 마이크볼륨막대기에
    변화가 없으면, 다른 마이크장치를 선택합니다.

  • 05.비디오 연결확인

    사용할 [비디오]장치를 선택한 후 정상 작동하면 다음을 클릭합니다.

  • 06.장비세팅 완료

    모든 장치가 정상적으로 설치되었는지 확인하고 다음을 클릭하여 설치를 마칩니다.

정상 설치 테스트

실제 수업환경과 동일한 상태에서 화상 및 음성대화에 이상이 없는 지 확인 점검하는 테스트입니다.
만약, 설치 순서대로 했음에도 여전히 정상적으로 동작하지 않으면 고객센터(1544-0159)로 연락하여 원격지원을 받으시기 바랍니다.

STEP
#01

모바일 화상강의실 설치방법

먼저, 스마트폰용 이어폰을 준비해 주세요.

화상강의실 설치버튼을 클릭하여 이티폰 화상앱을 다운받은 다음 설치하시면 됩니다.

  • 01.화상강의실설치 클릭

    이티폰 모바일 화면 메뉴의 [모바일화상강의실 설치]버튼을 클릭합니다.

  • 02.화상솔루션 설치

    스마트폰의 운영체제에 맞는 아이콘을 클릭하여 파일을 다운로드합니다.

iOS용(아이폰/패드) 기기는설정 > 일반 > 기기관리(프로파일) -> SAEHACOMMS.CO.LTD를 신뢰한다고 체크를 해주셔야 강의실 설치가 가능합니다.

STEP
#02

모바일 화상강의실 입장 방법과 자동녹화

  • 레벨테스트 입장 방법

    예약시간 5분전부터 홈페이지 우측 상단의 [화상테스트 강의실입장]버튼이
    활성화됩니다. 이 버튼을 클릭하여 강의실로 입장해 계시면
    강사님이 예약시간에 강의실로 입장하여 테스트가 진행됩니다.

  • 수강생 입장 방법

    예약시간 5분전부터 내강의실의 [화상강의실 입장]버튼이
    활성화됩니다. 이 버튼을 클릭하여 강의실로 입장해 계시면
    강사님이 예약시간에 강의실로 입장하여 수업이 진행됩니다.

자동 녹화 안내

이티폰의 진행된 모든 레벨테스트와 화상수업은 자동으로 녹화됩니다. 다만 고화질 녹화를 위하여 sc파일로 녹화되므로 모바일에서는 재생이 되지 않습니다.
